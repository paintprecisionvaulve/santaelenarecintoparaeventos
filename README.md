# Santa Elena · Jardín de Eventos

Sitio web de **Santa Elena Recinto para Eventos** — recinto de lujo para bodas y eventos en **San Miguel de Allende, Guanajuato, México**.

Es un sitio de **una sola página, todo en un solo archivo**: `index.html` ya incluye el diseño, los textos en español y **las 87 fotos del recinto** (incrustadas). No depende de carpetas ni de internet para las imágenes. Por eso son solo **2 archivos** y GitHub ya no se queja de “demasiados archivos”.

---

## 👀 Verlo en tu computadora

Haz **doble clic en `index.html`**. Se abre en tu navegador. Eso es todo.

---

## 🌐 Publicarlo gratis (elige una opción)

### Opción A — GitHub Pages
1. Crea un repositorio nuevo en https://github.com/new
2. Sube **solo estos 2 archivos**: `index.html` y `README.md`
   (botón **Add file → Upload files**, arrástralos y haz **Commit**).
3. En el repo: **Settings → Pages**.
4. En *Branch* elige **main** y la carpeta **/(root)** → **Save**.
5. En ~1 minuto tu sitio estará en: `https://TU-USUARIO.github.io/NOMBRE-DEL-REPO/`

> Importante: el archivo **debe llamarse `index.html`** para que GitHub Pages lo muestre como página principal.

### Opción B — Netlify Drop (lo más rápido, sin cuenta de GitHub)
1. Entra a https://app.netlify.com/drop
2. Arrastra el archivo `index.html` a la página.
3. Listo: te da una URL pública al instante. (Puedes conectar tu dominio después.)

### Opción C — Vercel
1. Sube los 2 archivos a un repositorio de GitHub (Opción A, pasos 1–2).
2. En https://vercel.com → **Add New → Project** → importa el repo → **Deploy**.

---

## 🔗 Conectar tu dominio (santaelenasma.com)

- **GitHub Pages:** Settings → Pages → *Custom domain* → escribe tu dominio y agrega el registro DNS que te indique.
- **Netlify / Vercel:** Domain settings → Add domain → sigue las instrucciones de DNS.

## 📝 Antes de publicar (recomendado)

1. **Reseñas reales.** La calificación **4.9 (128 reseñas)** y los testimonios son de muestra; reemplázalos por reseñas reales de Google.
2. **Pin del mapa.** El mapa apunta a la dirección por texto; verifica que el punto exacto coincida con la ubicación real del recinto en Google Maps.
3. **Tipografías.** Las fuentes (Cormorant Garamond + Jost) se cargan desde Google Fonts cuando hay internet. Sin conexión, el navegador usa tipografías del sistema (el sitio se ve bien igual).
